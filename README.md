# DentAI — AI Dental Clinic Landing Page

A modern landing page concept for an AI-powered dental clinic experience.

**Live site:** [dentaii.netlify.app](https://dentaii.netlify.app)

## About

DentAI is a demo/concept project showing what an AI-enhanced dental clinic 
website could look like. Built to pitch modern web solutions to local dental 
clinics across Punjab, India.

## Features

- Clean, modern UI designed for dental clinics
- AI-themed branding and copy
- Mobile responsive design
- Fast load, no bloat

## Tech Stack

- HTML / CSS / JavaScript
- Tailwind CSS (CDN)
- Netlify (deployment)
